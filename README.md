🎮 3D Game Project

📌 Overview
This is a 3D game developed using modern game development techniques. 
The game provides an immersive experience with interactive gameplay, smooth controls, and engaging mechanics.

🚀 Features
🧍‍♂️ 3D Character Movement (Walk / Run / Jump)
🎯 Shooting / Interaction System 
🗺️ Open Environment / Levels
🎥 Camera Control (First Person / Third Person)
🔊 Sound Effects & Background Music

🛠️ Built With
Game Engine: Godot
Language: GDScript

🎮 Controls
| Action       | Key        |
| ------------ | ---------- |
| Move Forward | W          |
| Move Back    | S          |
| Move Left    | A          |
| Move Right   | D          |
| Jump         | Space      |
| Shoot        | Left Click |
| Pause        | Esc        |
